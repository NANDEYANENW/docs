# サバイバル鯖

# はじめにすること

ようこそ、なんでやねんｗ鯖へ！ここでは、サバイバル生活を快適に始めるためのToDoリストを紹介します。  

### 1. 鯖のルール確認

まず、[鯖のルール](http://docs.nandeyanen.click/rule/#minecraft%E3%81%AE%E3%83%AB%E3%83%BC%E3%83%AB)と、[サバイバル鯖](http://docs.nandeyanen.click/rule/#%E3%82%B5%E3%83%90%E3%82%A4%E3%83%90%E3%83%AB%E3%82%B5%E3%83%BC%E3%83%90%E3%83%BC%E3%81%AE%E7%89%B9%E5%88%A5%E3%83%AB%E3%83%BC%E3%83%AB)のみ適用されるルールを確認しましょう。

### 2. 鯖のみんなに挨拶
チャットで他のプレイヤーに挨拶しましょう。鯖の雰囲気に慣れるためにも、積極的にコミュニケーションを取ることが大切です。

### 3. 職業に就く
なんでやねんｗ鯖には、様々な職業が存在します。農民、鉱山師など、自分の興味やプレイスタイルに合った職業を選びましょう。  
職業の選択コマンド: `/jobs borwse`<br>
<img src="/image/survival1.gif" width="1000">

### 4. 街に入るor街を作る
すでに存在する街に入るのも、自分で街を作るのも自由です。  
街に入る場合は、街のルールを守りましょう。

#### 街に入る場合

街のチャットで受け入れてくれる街を探しましょう。  
町長と連絡を取り、税金やルールについて確認しましょう。  
街のルールを守り、他の住民と協力して街を盛り上げましょう。  

#### 街を作る場合

安全な場所を選び、街の建設場所を決めましょう。  
街の名前、ルールなどを決めましょう。  
他のプレイヤーに協力を呼びかけ、一緒に街を作りましょう。  

### その他

チャットで他のプレイヤーと交流しましょう。  
分からないことがあれば、鯖のスタッフに聞いてみましょう。  
自分のペースで、マインクラフトライフを楽しんでください。  

### 注意事項

このToDoリストはあくまで参考です。  
鯖の状況やプレイスタイルによって、必要な行動は異なります。

# 各プラグインの使い方

### AureliumSkills  
#### 説明
スキルと経験値の管理を行うプラグイン。  
プレイヤーが特定のアクティビティを行うと、それに応じて報酬が得られる。  

#### コマンド
##### `/skills`:  
スキルの情報を表示する。  
##### `/skills top [スキル名] [ページ数]`:  
スキルランキングを表示する。  
##### `/skills top`:  
全スキルのランキングを表示する。

### BlockLocker  

#### 説明
チェストやドアなどのブロックをロックし、他のプレイヤーからアクセスを防ぐプラグイン。  
<img src="/image/survival2.gif" width="1000">

#### 機能
##### `[保護]`:  
ドアや、かまど、チェスト等の保護したいブロックに対して、看板を設置して保護する。 
##### `[編集]`:  
看板を右クリックして編集できます。次の行に他の人の名前を記入すると記入した人も保護ブロックに触れるようになる。  
##### `[全員]`:  
看板の編集で、`[全員]`というオプションを記入すると、全員対象のブロックを操作することができるようになる。
##### `[タイマー]`:  
看板の編集で`[タイマー:数字]`というオプションを記入するとその数字に応じて自分でクローズするようになる。  

### iConomy

#### 説明
仮想通貨(お金)の経済プラグイン。プレイヤーはお金を持ち、他のプレイヤーとの取引や経済活動を行います。  

#### コマンド
##### `/money`:  
所持金を表示する。  
##### `/pay [プレイヤー名] [数]`:  
他のプレイヤーに送金する。  
##### `/money top`:  
所持金ランキングを表示する。

### Jobs

#### 説明
プレイヤーが異なるジョブに参加し、それぞれのジョブで経験値を得ることができるプラグイン。各ジョブには異なるスキルや報酬がある。

#### コマンド
##### `/jobs`:  
現在のジョブと経験値を表示する。  
##### `/jobs join [ジョブの名前]`:  
ジョブに参加する。  
##### `/jobs leave [ジョブの名前]`:  
ジョブから脱退する。

### Towny

#### 説明
プレイヤーが町や国を作成し、土地を保護するためのプラグイン。プレイヤーは協力して大きなコミュニティを形成できる。
土地の取得ややプレイヤーの加入などの機能が提供されている。

#### コマンド
##### `/towny price`:  
町、国の運営の維持費などを表示する。  
##### `/towny time`:  
次回の税収のタイミングを把握する。  
##### `/towny top residents all`:  
市民数ランキングを表示する。  
##### `/town`:  
自分が在籍している街の情報を確認する。  
##### `/town [町の名前]`:  
指定した町の情報を確認する。  
##### `/town list`:  
街の一覧を表示する。  
##### `/town new [町の名前]`:  
新しい町を作成する。  
!!!primary 
この操作には、500NANDEのコストが必要です。
!!!
##### `/town add [プレイヤー名]`:
指定したプレイヤーを自分の町に招待する。
##### `/town kick [プレイヤー名]`:
指定したプレイヤーを自分の町から追い出せる。
##### `/town leave`:
在籍している町から脱退する。
##### `/town spawn`:
在籍している町のスポーン地点にテレポートする。
##### `/town spawn [町の名前]`:
指定した町のスポーン地点にテレポートする。
##### `/town claim`:
土地を取得することがする。  
!!!primary 
この操作には、5000NANDEのコストが必要です。
!!!  
:::danger 注意
この操作には、土地を取得する権限が必要です。
:::
##### `/town claim outpost`:
町から離れた場所の土地を確保する。    
!!!primary 
この操作には10000NANDEが必要です。
!!!
##### `/town unclaim`:
取得した土地を手放す。
##### `/town withdraw [金額]`:
指定し分のお金を町の銀行から引き出すこ。
##### `/town deposit [金額]`:
指定した分のお金を町の銀行に預ける入れる。
##### `/town outpost`:
アウトポスト（町から離れた拠点）にテレポートする。
##### `/town outpost [番号]`:
指定した番号のアウトポスト（町から離れた拠点）にテレポートする。
##### `/town ranklist`:
市民一覧と住民のランクを表示する。
##### `/town reslist`:
住民の一覧を表示する。
##### `/town set mayor [プレイヤー名]`:
指定したプレイヤーに市長を譲る。
##### `/town set homeblock`:
今いるチャンクを町のホームブロックに変える。
##### `/town set spawn`:
ホームブロックチャンク内のスポーン一を今いる場所に設定する。
##### `/town set name [名前]`:
町の名前を指定した名前に変更する。
##### `/town set taxes [数値]`:
税金を設定する。
##### `/town toggle [項目]`:
指定した項目のON/OFFを切り替える。
項目一覧:  
```
explosion (爆発)
fire (延焼)
mobs (敵MOBのスポーン)
public (公開設定)
open (参加受け入れ)
```
##### `/plot claim`:
販売中の小区画を購入する。
##### `/plot fs [金額]`:
今いるチャンクの小区画を指定した金額で販売する。
##### `/plot set shop`:
小区画を商店区画に設定する。
##### `/plot set embassy`:
小区画を大使館区画に設定する。
##### `/plot set arena`:
小区画をアリーナ区画に設定する。
##### `/resident`:
自分の情報を確認する。
##### `/resident [プレイヤー名]`:
指定したプレイヤーの情報を確認する。
##### `/resident tax`:
在籍している町の税金を確認する。
##### `/nation`:
加盟している国の情報を確認する。
##### `/nation online`:
オンラインの国民を確認する。
##### `/nation withdraw [金額]`:
指定した分のお金を国の銀行から引き出す。
##### `/nation deposit [金額]`:
指定した分のお金を国の銀行に預け入れる。
##### `/nation new [名前]`:
自分の町を首都に建国する。  
!!!primary 
この操作には5000NANDEのコストが必要です。
!!!
##### `/nation add [町名]`:
指定した町を自分の国に招待する。
##### `/nation kick [町名]`:
指定した町を自分の国から追い出す。
##### `/nation ally add [国名]`:
指定した国と同盟関係になる。
##### `/nation ally remove [国名]`:
指定した国の同盟関係を解除する。
##### `/nation enemy add [国名]`:
指定した国と敵対関係になる。
##### `/nation enemy remove [国名]`:
指定した国との敵対関係を解除する。
##### `/nation set king [プレイヤー名]`:
指定したプレイヤーに奥の座を譲る。
##### `/nation set capital [町名]`:
指定した町に首都を変更する。
##### `/nation set taxes [数値]`:
税金を設定する。
##### `/nation set name [名前]`:
指定した名前に国の名前を変更する。
##### `/town delete`:
町を削除する。  
:::danger 注意
町の中にある建造物はすべてリセットされます。
:::
##### `/nation delete`:
国を削除する。
